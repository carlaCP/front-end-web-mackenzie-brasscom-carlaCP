HTML Listas:

Comando de listagem: <li> </li>

    Listas Desordenadas:

    Exemplo:

    <ul>
  • <li> Cafe </li>
  • <li> Chá </li>
  • <li> Leite </li>
</ul>
    Listas Ordenadas:

    Exemplo:

    <ol>
  1. <li> Cafe </li>
  2. <li> Chá </li>
  3. <li> Leite </li>
</ol>
