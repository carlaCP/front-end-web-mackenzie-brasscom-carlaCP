HTML Imagens

    HTML images sao definidadas por </img> tag.

    Usar source file (src), texto alternativo (alt), width, and height sao atributos que podem ser fornecidos

    Exemplo de formato :

    <img> src="w3schools.jpg" alt="W3Schools.com" width="104" height="142" </img>
