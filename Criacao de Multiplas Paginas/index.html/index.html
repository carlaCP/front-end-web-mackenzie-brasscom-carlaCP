HTML

HTML é linguagem padrão para criação de páginas web. É a abreviação para Hyper Text Markup Language (Linguagem de Marcação de Hipertexto. É composto por elementos que são formados por tags e indicam para o navegador como os conteúdos devem ser exibidos para o usuário.

A responsabilidade do HTML é permitir a estruturação do conteúdo da nossa página, bem como atribuição de significado (semântica) para os mesmos. Por exemplo, com HTML eu consigo indicar que um conteúdo é um título, um parágrafo, uma lista, um trecho importante, etc.

Abaixo, a explicação sobre o funcionamento de algumas tags do HTML.

  • Tags de Titulo (<h1>,<h2>,<h3>,<h4>,<h5>,<h6>)
  • Tag de Paragrafo (<p>)
  • Tag de Imagem (<img>)
  • Tag de Link (<a>)
  • Tags de Lista (<ul>,<ol>,<li>)
